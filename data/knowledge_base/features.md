# ProTaskFlow - Features Guide

## Core Features

### Task Management

#### Creating Tasks
- Click "Add Task" or press `T`
- Set title, description, and assignee
- Add due dates, priorities, and labels
- Attach files up to 100MB (Pro+)
- Add subtasks for complex items

#### Task Views
- **List View**: Traditional list with sorting options
- **Board View**: Kanban-style drag-and-drop
- **Calendar View**: See tasks by due date
- **Timeline/Gantt**: Visualize project schedules
- **Table View**: Spreadsheet-like bulk editing

#### Task Properties
- **Priority**: Low, Medium, High, Urgent
- **Status**: Customizable workflow stages
- **Labels**: Color-coded tags
- **Custom Fields**: Text, numbers, dates, dropdowns (Pro+)
- **Dependencies**: Link related tasks

---

### Project Management

#### Project Templates
Choose from 20+ templates:
- Marketing Campaign
- Software Development Sprint
- Event Planning
- Content Calendar
- Bug Tracking
- OKR Tracking

#### Project Views
- Overview dashboard with progress
- Activity feed for all updates
- Files library
- Team member list
- Project settings

#### Portfolios (Business+)
Group related projects together:
1. Go to **Portfolios** in sidebar
2. Click "New Portfolio"
3. Add projects to your portfolio
4. View combined progress and reports

---

### Collaboration

#### Comments & Discussions
- Comment on any task
- @mention team members
- React with emojis
- Thread replies for organized discussions
- Rich text formatting with markdown

#### Real-time Collaboration
- See who's viewing the same task
- Live cursor tracking
- Instant updates across all devices
- No refresh needed

#### Team Communication
- **Direct Messages**: 1-on-1 chat
- **Group Channels**: Team discussions
- **Project Channels**: Auto-created per project
- **Threads**: Keep conversations organized

---

### Time Tracking (Pro+)

#### Track Time
1. Open a task
2. Click the timer icon ⏱️
3. Start/stop tracking
4. Or enter time manually

#### Time Reports
- View by project, team member, or date range
- Export to CSV/Excel
- Billable vs non-billable time
- Integration with invoicing tools

---

### Reporting & Analytics (Pro+)

#### Built-in Reports
- **Workload**: See team capacity
- **Progress**: Project completion rates
- **Time**: Where time is spent
- **Productivity**: Tasks completed trends

#### Custom Dashboards
1. Go to **Dashboards**
2. Click "New Dashboard"
3. Add widgets: charts, lists, metrics
4. Share with team or keep private

---

### Automation (Pro+)

#### Rules
Automate repetitive tasks:
- When task is created → assign to specific person
- When status changes to Done → notify channel
- When due date passes → increase priority
- When label is added → move to project

#### Templates
Create task templates for recurring work:
1. Create a task with all details
2. Click ⋯ > "Save as Template"
3. Use template for new tasks

---

### Security Features (Business+)

#### Access Controls
- **Workspace Admin**: Full control
- **Project Admin**: Manage specific projects
- **Member**: Collaborate on assigned projects
- **Guest**: Limited access to specific items

#### Security Options
- Two-factor authentication (2FA)
- Single Sign-On (SSO) via SAML
- IP allowlisting
- Session timeout controls
- Audit logs for compliance

---

### Integrations

#### Popular Integrations
- **Slack**: Get notifications, create tasks from messages
- **Google Workspace**: Calendar sync, Drive attachments
- **Microsoft 365**: Teams, Outlook, OneDrive
- **GitHub/GitLab**: Link commits to tasks
- **Figma**: Embed designs
- **Zapier**: 3000+ app connections

#### API Access (Business+)
- RESTful API for custom integrations
- Webhooks for event notifications
- OAuth 2.0 authentication
- Rate limits: 1000 requests/hour
