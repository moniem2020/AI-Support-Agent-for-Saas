# ProTaskFlow - Troubleshooting Guide

## Common Issues & Solutions

### Login Problems

#### "Invalid credentials" error
1. Verify you're using the correct email address
2. Try resetting your password via "Forgot Password"
3. Check if Caps Lock is on
4. Clear browser cookies and try again
5. If using SSO, ensure your company SSO is active

#### Two-Factor Authentication (2FA) issues
1. Check that your device time is synchronized
2. Use backup codes if you've lost your authenticator
3. Contact support to reset 2FA: support@protaskflow.com

#### Locked out of account
After 5 failed login attempts, accounts are locked for 15 minutes. Wait and try again, or contact support for immediate unlock.

---

### Sync Issues

#### Tasks not syncing across devices
1. Check your internet connection
2. Force refresh: `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
3. Log out and log back in
4. Clear browser cache
5. Check our [status page](https://status.protaskflow.com)

#### Mobile app not syncing
1. Ensure you're on the latest app version
2. Check internet/WiFi connection
3. Force close and reopen the app
4. Re-login to the app

---

### Performance Issues

#### Slow loading times
1. Clear browser cache and cookies
2. Disable browser extensions (especially ad blockers)
3. Try a different browser (Chrome, Firefox, Safari recommended)
4. Check your internet speed
5. Large projects (1000+ tasks) may load slower - use filters

#### Browser crashes or freezes
1. Ensure you have at least 4GB RAM available
2. Close unnecessary tabs
3. Update your browser to the latest version
4. Disable hardware acceleration in browser settings

---

### Notification Issues

#### Not receiving email notifications
1. Check spam/junk folder
2. Add notifications@protaskflow.com to contacts
3. Verify email in **Settings** > **Notifications**
4. Check notification preferences are enabled
5. Some email providers delay messages - wait 5-10 minutes

#### Too many notifications
1. Go to **Settings** > **Notifications**
2. Customize which notifications you receive
3. Enable "Digest Mode" for daily summaries instead of real-time
4. Mute specific projects or conversations

---

### File Upload Issues

#### "File too large" error
- Free plan: Max 25MB per file
- Pro plan: Max 100MB per file
- Business/Enterprise: Max 500MB per file

To upload larger files, upgrade your plan or use our integration with Google Drive/Dropbox.

#### "Upload failed" error
1. Check your internet connection
2. Try a different file format
3. Ensure the file isn't corrupted
4. Clear browser cache
5. Try uploading from a different browser

---

### Integration Problems

#### Slack integration not working
1. Go to **Settings** > **Integrations** > **Slack**
2. Click "Reconnect"
3. Re-authorize the Slack workspace
4. Verify the correct channel is selected

#### Google Calendar not syncing
1. Check that the integration is connected
2. Verify calendar permissions
3. Wait up to 15 minutes for initial sync
4. Toggle the integration off and on

## Error Codes

| Code | Meaning | Solution |
|------|---------|----------|
| E001 | Network timeout | Check internet, retry |
| E002 | Authentication failed | Re-login |
| E003 | Permission denied | Contact workspace admin |
| E004 | Resource not found | Item may be deleted |
| E005 | Rate limited | Wait 1 minute, retry |
| E500 | Server error | Contact support |
